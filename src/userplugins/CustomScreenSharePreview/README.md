# Custom Screen Share Preview

Adds the ability to send a custom image as a preview for your screen share in Discord.

# Prerequisites

Follow the [Prerequisites](https://docs.vencord.dev/installing/#prerequisites) section from Vencord documentation.

# Installation

1. Clone [Vendicated/Vencord](https://github.com/Vendicated/Vencord) repository.
2. Create a folder named `userplugins` in the `<VENCORD_REPOSITORY>/src/`.
3. Clone or download this repository into the `userplugins` folder.
4. Follow [Vencord build instructions](https://docs.vencord.dev/installing/) to build Vencord with this plugin.
5. Once Vencord is built and injected, open Discord, go to `Settings` > `Plugins`, and enable the **CustomScreenSharePreview** plugin.

# Usage

1. Start sharing screen and enable the **Hide stream preview** checkbox while configuring the screen share.
2. Press the **Stream Preview Image** button. You can found it right before the **Settings** button in the bottom left corner of the screen.
3. Select any image you want to use as a preview and press **Upload** button.
